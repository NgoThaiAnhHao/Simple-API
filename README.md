# Product Order API

## Tech Stack
- Spring Boot
- Spring Data JPA
- Spring Validation
- Spring Web
- Spring Devtools
- Spring Security
- MySQL
- Swagger OpenAPI

## Features
- Product management
- Order management
- Authentication (JWT)

## API Documentation
- Swagger UI docx (Overview): https://editor.swagger.io/ 
- Swagger UI (Localhost): http://localhost:8080/swagger-ui.html

## Main Endpoints
- Products:
  + GET /api/products
  + GET /api/products/{id}
  + POST /api/products
  + PUT /api/products/{id}
  + PATCH /api/products/{id}
  + DELETE /api/products/{id}
- Orders:
  + GET /api/orders
  + GET /api/orders/{id}
  + POST /api/orders
